== Description ==

This plugin puts a customizable message bar at the top of your website that users can hide. You can type in your own message and link it to anything you would like. This is a great way to quickly get a message out to the users of your website. Sample ideas: show a coupon, contest, or limited time offer to the users of your website.

To use it, install the plugin and fill out the options on the Message Bar options panel under settings.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the 'message-bar' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Message Bar and fill out the options. 

== Frequently Asked Questions ==

= What do I do with this?  =

After you install, make sure to fill out all options under the Settings > Message Bar panel in the WordPress admin area.
